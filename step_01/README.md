## 딥봇 구조
딥봇 프레임웍의 구조는 크게 화면UI를 담당하는 프런트앤드 파트와 서버로직을 담당하는 백앤드로 구성되어 있다. 프런트앤드는 HTML, CSS, JS, 백앤드는 Flask 로 구현되어 있다.

폴더 구조는 다음과 같다.

```
/static
  /css
    style.css
  /images
    logo_w.png
    logo.png
/templates
  index.html
app.py
```
